# Прогнозирование кредитного дефолта

##  Описание проекта
Построение модели машинного обучения для прогнозирования невыполнения долговых обязательств по кредиту на основе данных о клиентах банка.

##  Технологии
- Python 3.8+
- Pandas, NumPy - обработка данных
- Matplotlib, Seaborn - визуализация
- Scikit-learn - машинное обучение
- Imbalanced-learn - балансировка классов (SMOTE)
- Google Colab / Jupyter Notebook

##  Данные
- Обучающая выборка: 7500 записей, 16 признаков
- Тестовая выборка: 2500 записей
- Целевая переменная: Credit Default (1 - дефолт, 0 - нет)

##  Результаты EDA
- Выявлен дисбаланс классов: 71.8% / 28.2% (соотношение 2.55:1)
- Обнаружены пропуски в 5 признаках (до 54%)
- Корреляционный анализ показал важность Credit Score (0.44)

##  Предобработка данных
- Преобразование текстовых значений в числовые
- Заполнение пропусков медианой/модой
- One-Hot Encoding категориальных признаков
- Масштабирование (StandardScaler)
- Балансировка классов (SMOTE и class_weight)

##  Моделирование

### Baseline модели (F1-score)
| Модель | F1-score |
|--------|----------|
| Logistic Regression | 0.663 |
| Decision Tree | 0.739 |
| Random Forest | 0.833 |
| Gradient Boosting | 0.772 |

### Лучшая модель
- **Алгоритм**: Logistic Regression с `class_weight='balanced'`
- **F1-score на валидации**: 0.5151
- **Статус**:  Требование F1 > 0.5 выполнено

### Сравнение алгоритмов (повышенный уровень)
| Реализация | F1-score |
|------------|----------|
| Самописная логистическая регрессия | 0.386 |
| Библиотечная (scikit-learn) | 0.523 |
| **Разница** | 0.137 |

##  Ключевые факторы дефолта
1. **Credit Score** (кредитный рейтинг)
2. **Current Loan Amount** (текущая сумма кредита)
3. **Annual Income** (годовой доход)
4. **Number of Credit Problems**
5. **Years of Credit History**
